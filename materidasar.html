HTML     Dasar

Struktur dasar HTML terdiri dari 4 bagian.

1. !DOCTYPE html : Bagian ini memperjelas jenis dokumen yang sedang ditampilkan oleh browser. Mendefinisikannya cukup penting agar dokumen HTML yang kita bangun bisa di-parsing dengan baik oleh semua browser.

2. Dokumen html : Menandakan kapan dan di mana dokumen html dimulai.

3. Bagian head : Bagian yang tidak terlihat oleh user, akan tetapi memiliki peran penting untuk menyiapkan berbagai informasi terkait halaman web, salah satu contohnya adalah untuk menuliskan judul halaman web.

4. Bagian body : Bagian di mana semua konten yang akan dilihat user ditulis.

Heading merupakan element atau tag HTML yang berfungsi untuk menunjukkan bagian penting pada halaman web. Element tag heading ini memiliki enam tingkatan yang berurutan yaitu h1, h2, h3, h4, h5, h6 yang bisa digunakan untuk menambah struktur ke halaman web.

Paragraf adalah kumpulan dari beberapa kalimat. Pada web, paragraf biasanya digunakan untuk menampilkan teks atau artikel. Paragraf pada HTML dibuat dengan tag < p>.
