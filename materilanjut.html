HTML     Lanjut

Link pada HTML dapa dibuat dengan tag < a>, kemudian tag ini harus memiliki atribut href untuk menentukan alamat URL tujuan dari link.
Elemen ini bisa diklik dan nanti akan membuka halaman lain sesuai alamat URL yang diberikan.

Tag < img> digunakan untuk menampilkan gambar pada suatu halaman HTML. Atribut src adalah singkatan dari source, merupakan atribut yang berisi alamat dari gambar yang akan ditampilkan. Alamat ini bisa relatif atau absolute.

Tag < iframe> pada dokumen HTML adalah sebuah tag yang berfungsi untuk menampilkan sebuah halaman web (HTML) di dalam halaman web yang lain.

List dibagi menjadi dua yakni ordered list dan unordered list. Ordered list dibuat dengan tag < ol>. Lalu, di dalamnya diisi dengan item-item yang akan dimasukkan ke dalam list. Item dibuat dengan tag < li> atau yang disebut dengan (list item).

Unordered list adalah list yang tak terurut yang menggunakan simbol-simbol pada item-nya. Unordered list dibuat dengan tag < ul> dan untuk item-nya dibuat juga dengan tag < li>.

Tabel merupakan struktur yang digunakan untuk menampilkan informasi dalam bentuk baris dan kolom. pada HTML, untuk membuat tabel menggunakan tag < table>. Biasanya tag < table> digunakan untuk menampilkan data yang struktur atau tampilan data dari database.
