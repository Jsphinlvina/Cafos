CSS



Struktur CSS


h1 ➜ Selektor

Selektor adalah kata kunci untuk memilih elemen HTML yang akan kita beri style.


{color: green; font-size:40px;} ➜ Deklarasi

Deklarasi adalah tempat kita menuliskan atribut atau properti CSS yang akan diberikan kepada elemen yang telah diseleksi.


(#) ID


Selector Id digunakan untuk memilih elemen tertentu. Id tidak boleh diawali oleh angka dan hanya dapat digunakan sekali

(.) Class


Digunakan untuk memilih elemen dengan kelas tertentu. Kita dapat menentukan elemen HTML tertentu yang harus terpengaruh oleh suatu kelas.

(*) Universal


Digunakan untuk memilih semua elemen HTML pada halaman


Komentar


Komentar CSS ditempatkan di dalam <style> elemen, dan dimulai dengan /* dan diakhiri dengan */

3 Cara menyisipkan style sheet


  • External style sheet
    • Simpan dengan ekstensi .css
    • Tidak boleh ada tag HMTL apapun
    • Tulis didalam <head> sintaks: <link rel = "stylesheet" href = "style.css">

  • Internal style sheet
    • Ditulis didalam <head>

  • Inline style
    • Menerapkan style unik untuk satu elemen HTML.

CSS Background Shorthand Property


Dari:


body {
background-color : #ffffff
background-image : url ("house.png");
background-repeat : no-repeat;
background-position : right top;
}

Menjadi:


body {
background: #ffffff url("house.pgn") no-repeat right top; }
